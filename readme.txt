Clover Online Orders to integrate the e-commerce in your website

== Description ==

You have a wordpress website and you use Clover to manage your commerce, you don"t want change your website to add the
online orders.

Our plugin is the best solution, with Clover online Orders you can add the e-commerce in less than 5 minute.

A few notes :

*   You should have a "Clover POS" to use this plugin.
*   You should install our app from the appmarket.
*   this plugin is created by Merchantech not Clover.com

Demo : http://test.merchantech.us


== Installation ==

How to install the plugin and get it working:


1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the OnlineOrders Name screen to configure the plugin
1. Add your token
